# Training plots

These PNGs are generated by **Cell 12** of `training/train_colab.ipynb`.

After training finishes in Colab, run:

```bash
cd /content/meta_scaler
git add docs/plots
git commit -m "Add training plots"
git push
```

Expected files:
- `reward_curve.png` — per-episode reward + 10-ep moving average
- `loss_curve.png` — GRPO loss over episodes
- `survival_curve.png` — episode length (60 = survived full year)
- `metrics.json` — raw training metrics
